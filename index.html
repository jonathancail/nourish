Step 1 of 2 — Firebase Setup
1
Create a free Firebase project
Go to console.firebase.google.com → Add project → give it a name → disable Google Analytics (optional) → Create project.
2
Enable Google Sign-In
In your project: Authentication → Get started → Sign-in method → Google → Enable → Save. Add your email as a test user under Authentication → Users if prompted.
3
Create Firestore database
Firestore Database → Create database → Start in test mode → Next → Enable. Test mode gives you 30 days of open access — you can tighten rules later.
4
Register a Web app & paste the config
Project settings (⚙) → Your apps → Add app → Web (</>) → give it a nickname → Register. Copy the firebaseConfig object and paste it below.
